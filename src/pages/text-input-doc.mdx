# Text Input 

######

## Overview 

Text Input are used to create interactive controls for web-based forms in order to accept data from the user. Users could send their data in 
wide varety of types. Although we could describe input as anything user send to the databases, we limited the ontology of the Text Input to an input with the string type value.
Therefore, date picker, file uploader and any other input formats doesn't include by this component.  

## Usage

A `<TextInput>` with `leftIconName`, `rightIconName`, `name`, `title`, and `sizeType` has been represented below.

### Live Demo 

```js

// for the matter of usage in React, we need import the component like below:
// import { TextInput } from '@undertheground/react';

<TextInput
  name="Password"
  leftIconName="password"
  placeholder="Exp: abas0098"
  sizeType="medium"
  title="Password"
  rightIconName="add"
/>


```
